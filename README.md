# Databricks-movies

Diseñé e implementé una arquitectura de datos tipo Medallion (Bronze, Silver, Gold) en Azure Databricks utilizando Apache Spark y Delta Lake, con almacenamiento en Azure Data Lake Gen2. A partir del dataset de películas de Database Star, realicé la ingesta, transformación y análisis de grandes volúmenes de datos estructurados y semiestructurados (CSV, JSON). Configuré clusters, optimicé cargas incrementales, apliqué técnicas de particionado y caché, y conecté la capa de reporte con Power BI. Adquirí experiencia en Spark SQL, manejo de Azure Storage, Databricks File System (DBFS), monitoreo de jobs y colaboración mediante notebooks en Python (PySpark) y SQL
